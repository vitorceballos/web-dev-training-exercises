# Cookies and New Window

This activity demonstrates how to use cookies and manipulate browser windows using basic JavaScript techniques.

It was developed as part of the "Client-Side Web Development" module during my Higher Technical Education in Web Application Development (EQF Level 5).

## Description

This activity focuses on handling browser cookies and interacting with new windows through JavaScript. It includes:

- Requesting and storing the user's name in a cookie
- Displaying a personalized greeting on future visits
- Opening a new browser window with custom content and dimensions
- Managing user input with basic validation

## Features

- Cookie creation and retrieval using `document.cookie`
- Use of `window.open()` to create a custom popup
- DOM manipulation to interact with forms and display messages
- Basic CSS for layout and visual structure

## Files

- `index.html`: HTML structure with input form and button
- `cookies.js`: JavaScript logic for managing cookies and the new window
- `styles.css`: Styles for the form and overall page layout

## Technologies Used

- HTML
- CSS
- JavaScript
